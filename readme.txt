== Description ==

This widget accommodates different sizes of image ads. With this plugin you can create unlimited number of ads inside your WordPress widget. There are several smart options provided to customize ads for your needs.

== Installation ==

1. Upload Ads-Widgets-Easy.zip to plugins via WordPress admin panel or upload unzipped folder to your wp-content/plugins/ folder
2. Activate the plugin through the "Plugins" menu in WordPress
3. Go to Appearance -> Widgets to add and setup the widget
